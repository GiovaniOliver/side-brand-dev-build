# SMG Side Brands - Development

## Overview

Development workspace for all side brand projects. Uses a standardized brand system template and tech stack for consistent builds across all brands.

## Available Commands

This workspace has custom slash commands in `.claude/commands/`:

| Command | Purpose |
|---------|---------|
| `/create-brand` | Initialize a new brand from template |
| `/build-website` | Build a brand website |
| `/deploy-brand` | Deploy a brand to production |
| `/launch-brand` | Full brand launch sequence |
| `/check-brand-status` | Check build/deploy status |
| `/full-pipeline` | Run complete brand pipeline |
| `/brand-intake` | New brand intake process |

## Brand Template

`1. Brand System Template & Tech Stack Template/` contains the standard template for all side brand projects.

## Active Projects

Projects with active development repos in this folder. Each brand follows the standardized template structure.

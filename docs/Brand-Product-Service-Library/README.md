# Brand Product Service Library

This folder centralizes the cross-brand product, monetization, affiliate, and third-party service review docs for the documentation-only repo.

Use these files in order:

1. `PRODUCT_SERVICE_DOC_INDEX.md`
   Exact markdown inventory for the product and monetization docs that exist in this repo.
2. `BRAND_PRODUCT_CATALOG.md`
   Quick review of what each brand is currently selling or planning to sell.
3. `WHITE_LABEL_THIRD_PARTY_SERVICE_MATRIX.md`
   Working matrix for which brands need white-label, affiliate, fulfillment, marketplace, or service partners.

Notes:
- This repo is docs-only, so the library references `brands/`, `docs/`, and `_Working-Docs/` only.
- Azure Tide Wine CO is the renamed brand previously stored as `DolphinWine`.

# Resume AI Feedback - Brand Info

Resume AI Feedback is an AI-assisted career tool that analyzes resumes, provides structured improvement guidance, and can expand into job matching and premium rewrite workflows.

## Current Product Direction

- Primary model: freemium SaaS or productized career tool
- Core audience: job seekers and career switchers
- Main value: fast AI-powered feedback with a path to better resumes and stronger job targeting

## Revenue Direction

- Premium analysis or rewrite credits
- Subscription for advanced feedback and job-match features
- Resume template and toolkit sales
- Career-tool affiliate partnerships
- B2B licensing for schools, coaches, or workforce programs

## Current State

- The application codebase is active in the main dev workspace
- Internal product docs exist, but public brand-system docs are still thin
- The blog and public acquisition layer are not yet built out in this repo

## Main Missing Pieces

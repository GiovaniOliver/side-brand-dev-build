## Instructions for Uploading to Sanity Studio

### Step 1: Access Sanity Studio
1. Navigate to: http://localhost:3000/studio (if dev server is running)
2. Or deploy and access at: https://your-domain.com/studio

### Step 2: Create Author (if not exists)
1. Click "Author" in the left sidebar
2. Create a new author with name, bio, and image

### Step 3: Create Categories (if not exists)
Create the following categories:
- AI Tools
- Tutorials
- Guides
- ChatGPT
- Midjourney
- Voice AI
- Image Generation
- Content Creation
- Automation
- Programming

### Step 4: Upload Images
For each blog post:
1. Copy the corresponding image from the images folder
2. When creating the post, upload the image as the "Main Image"
3. Add alt text describing the image

### Step 5: Add Blog Posts
For each blog post listed above:

1. Click "Post" in Sanity Studio sidebar
2. Click "Create new Post"
3. Fill in the following fields:
   - **Title**: (from mapping above)
   - **Slug**: Auto-generate from title or manually create SEO-friendly slug
   - **Excerpt**: (from mapping above - use first 200 chars)
   - **Author**: Select the author you created
   - **Main Image**: Upload the corresponding image from mapping
   - **Alt Text**: Descriptive text for the image
   - **Categories**: Select relevant categories
   - **Published At**: Set to current date/time or schedule
   - **Body**: Copy and paste the markdown content from the file
   - **Featured Post**: Check this for the 4 main guides
   - **Read Time**: (from mapping above)
   - **SEO Keywords**: (from mapping above)
   - **Content Type**: `blog`, `workflow`, or `tutorial` based on mapping

4. Click "Publish" when done

### Step 6: Convert Markdown to Block Content

The blog post body needs to be converted from markdown to Sanity's block content format. You have two options:

**Option A: Manual Formatting in Sanity Studio**
- Copy the markdown content
- Use Sanity Studio's rich text editor to format:
  - Headings (H2, H3, etc.)
  - Bold and italic text
  - Code blocks
  - Lists
  - Blockquotes

**Option B: Use a Markdown to Block Content Converter** (recommended)
- Install: `npm install @sanity/block-content-to-markdown`
- Or use Sanity's import tools

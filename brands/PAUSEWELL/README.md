## 🚀 How to Build a New Brand

### Step 1: Understand the System

**Read these 3 files in order:**

1. **`.claude/MASTER_BRAND_STATUS_AND_SYSTEM.md`**
   - See status of all brands
   - Understand the complete system
   - Choose which brand to build

2. **`00-MASTER-BRAND-SYSTEM-GUIDE.md`**
   - Complete 4-phase building process
   - Step-by-step instructions
   - Tech stack overview

3. **`Brand-Progress-Tracking/README.md`**
   - How to track your progress
   - Update schedules
   - Tracking templates

---

### Step 2: Set Up Tracking

```bash
# 1. Go to tracking folder
cd Brand-Progress-Tracking/Active-Brands

# 2. Create folder for your brand
mkdir "[Brand Name]"

# 3. Copy tracking templates
copy ../Templates/BRAND_PROGRESS_TRACKER.md "[Brand Name]/"
copy ../Templates/GOALS_AND_RESULTS_TRACKER.md "[Brand Name]/"
copy ../Templates/WEEKLY_TASK_TRACKER.md "[Brand Name]/"

# 4. Fill in the templates
```

---

### Step 3: Complete Planning

**If brand folder doesn't exist:**
1. Create brand folder at root
2. Copy templates from `1. Brand System Template & Tech Stack Template/`
3. Fill in all brand information

**If brand folder exists:**
1. Review existing files
2. Complete any missing documents
3. Update outdated information

---

### Step 4: Build Website

Follow the detailed guide in:
**`00-MASTER-BRAND-SYSTEM-GUIDE.md`** → Phase 2: Build

Tech stack is documented in:
**`UNIVERSAL_TECH_STACK_GUIDE.md`**

---

### Step 5: Launch

Follow deployment guides:
- `MASTER_DEPLOYMENT_CHECKLIST.md`
- `VERCEL_DEPLOYMENT_CHECKLIST.md`
- `POST_DEPLOYMENT_CHECKLIST.md`

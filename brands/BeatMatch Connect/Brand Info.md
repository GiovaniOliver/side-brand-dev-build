# BeatMatch Connect - Brand Info

BeatMatch Connect is a marketplace-style music collaboration platform built to connect rappers, singers, and producers through audio-aware matching, messaging, and collaborative workflows.

## Current Product Direction

- Primary model: two-sided marketplace plus subscription
- Core user groups: artists and beat producers
- Main value: better match quality, faster collaboration, and platform-led transactions

## Revenue Direction

- Artist subscriptions
- Producer subscriptions
- Transaction fees on collaborations or beat deals
- Featured placement or boosted discovery
- Service upsells such as custom work or studio support

## Current State

- Streamlit version is functional
- Next.js migration is in progress
- Public brand docs in this repo are still light compared with the main dev workspace

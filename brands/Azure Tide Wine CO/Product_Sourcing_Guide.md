# Azure Tide Wine CO - Product Sourcing Guide

## Licensing Requirements (CRITICAL - DO FIRST)

### Federal (TTB)
- **Basic Permit** (FAA Act) - Form TTB 5100.24 at [ttb.gov](https://www.ttb.gov/online-services/applications) - FREE
- **COLA (Certificate of Label Approval)** - Every wine label needs TTB approval at [COLAs Online](https://www.ttb.gov/alfd/certificate-of-label-aproval-cola) - FREE ($50 if using filing service)
- Approval: 5-20 business days

### State
- **Direct Shipper Permit** needed in each state you ship to (~47 states allow DTC wine shipping)
- Your fulfillment partner (e.g., KDM, Wineshipping) typically holds these permits for you
- Reference: [Sovos ShipCompliant State Rules](https://sovos.com/shipcompliant/resources/dtc-wine-shipping-rules-by-state/)

### Compliance Software
- **Sovos ShipCompliant** - [sovos.com/shipcompliant](https://sovos.com/shipcompliant) - Industry standard for real-time state compliance
- **DRINKS ASSURE** - [drinks.com/assure](https://drinks.com/assure) - Shopify-native compliance app for checkout

## Recommended Launch Strategy

### Phase 1 - Launch (Zero Inventory)
- Partner: **KDM Global Partners** (single-vendor: production + compliance + DTC drop ship)
- Store: **Shopify + DRINKS ASSURE** for compliance at checkout
- Azure Tide Wine CO operates as brand/marketing layer only

### Phase 2 - Scale (Owned Inventory)
- Move to owned inventory at **Wineshipping.com** or **WineCare Logistics**
- Build wine club through **Cellars Wine Club Fulfillment**
- Add **Sovos ShipCompliant** as compliance backbone

### Licensing Priority Order
1. Establish LLC
2. Apply for TTB Basic Permit (free, online)
3. Get TTB COLA approval for label design
4. Verify state DTC permits via Sovos
5. Engage fulfillment partner (they hold most state permits)
